## Project Description
This project contains two JavaScript applications: an Interactive Calculator and a Student Grade Tracker. The calculator performs basic arithmetic operations, while the Grade Tracker stores student data using arrays and objects and determines whether students pass or fail.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Perform addition, subtraction, multiplication, and division.
- Track student names and grades.
- Interactive and easy-to-use interface.

## How to Run
1. Clone this repository.
2. Open the `index.html` file in your web browser.

## Lessons Learned
- How to use variables and data types.
- How to create and call functions.
- How to use arrays and objects.
- How to manipulate the DOM with JavaScript.
- How to use loops and conditional statements.

## Challenges Faced
One challenge was making the calculator display the correct results and changing button styles with CSS. I solved it by using JavaScript functions and proper CSS selectors.
